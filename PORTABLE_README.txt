SkinWatcher Portable
====================

1. Keep the entire SkinWatcher folder together.
2. Double-click SkinWatcher.exe.
3. Do not move only the EXE out of this folder.

Python is not required. Chromium and all application dependencies are included.
Your settings and watcher state are stored beside SkinWatcher.exe.

Windows may show a SmartScreen warning because this application is not
code-signed. If you trust the source of this ZIP, choose "More info" and then
"Run anyway".
